1) structure of table =>
    1. =>table
        ->caption
        ->tr - table row

    2. =>table
        ->thead - it covers upper section of table
            tr>th
        ->tbody - it covers middle section of table
            tr>td
        ->tfoot - it covers bottom section of table
            tr>th

List of property in table - tr,th,td =>
    -> align - left , right , center
    -> valign - top , middle , bottom
    -> border
    -> cellpadding - it apply space inside th and td
    -> cellspacing - it apply space outside of th and td
    -> width - in px or %
    -> height - in px or %
    -> bgcolor - use to set background color
    -> colspan - it use to combine space of 2 or more th and td
    -> rowspan - it use to combine space of 2 or more tr
